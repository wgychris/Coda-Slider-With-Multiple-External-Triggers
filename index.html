Coda Slider w/Multiple External Triggers

By: Myke Preuss

The purpose of this code is to allow you to have multiple external triggers on a page that control the same Coda Slider. The external triggers communicate with each other and add a class called "current" to the trigger which was clicked. The "current" class is what's adding the text-decoration: underline; but you could style it any way you want in your CSS.

The files you'll need for the Coda Slider are here and you can keep your jQuery set to the most recent in the HTML mark up.

Your external controls must be in a ul with a class of "slideControls" for this code to work.
